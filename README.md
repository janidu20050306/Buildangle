## ✨ Features

- **Modern Design** - Clean, professional construction theme with orange accent colors
- **Responsive Layout** - Fully optimized for mobile, tablet, and desktop
- **Admin Portal** - Secure owner-only access to manage projects
- **Interactive Chatbot** - AI-powered assistant for customer inquiries
- **Project Showcase** - Beautiful portfolio with filterable projects
- **Multi-page Structure** - Home, About, Services, Projects, Blog, Contact, Quote
- **Smooth Animations** - Page transitions and scroll effects
- **SEO Optimized** - Meta tags and Open Graph support

## 🚀 Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/janidu20050306/Buildangle.git

# Navigate to project
cd builangle

# Install dependencies
npm install

# Start development server
npm run dev
```

### Build for Production

```bash
npm run build
npm start
```
