## 🏛️ Strategic Vision
- **Institutional Authority**: Established via 17+ years of excellence.
- **Cinematic Experience**: Staggered motion reveals and 3D integration.
- **Conversion Optimization**: Multi-step quote engine and VIP inquiry routing.

## 🛠️ Technology Stack
- **Frontend**: Next.js 15, React 19, TypeScript, Tailwind CSS v4.
- **Animations**: Framer Motion 12, GSAP 3.15, React Spring.
- **3D Engine**: Three.js (r164), React Three Fiber, Spline/Sketchfab ready.
- **Backend**: MongoDB Atlas (Mongoose), Server Actions, API routes.
- **Performance**: <1.8s LCP, Image optimization (WebP/AVIF), Edge Caching.

## 📁 Project Architecture
- `/app`: App Router with dynamic routes for Projects and Blog.
- `/components`: Custom premium UI library using `shadcn/ui` principles + Framer Motion.
- `/lib`: Database connectivity, analytics, and app-wide constants.
- `/models`: Mongoose schemas for Projects, Testimonials, and Inquiries.
- `/styles`: Global theme definitions with Tailwind CSS v4 CSS variables.

## 🚀 Getting Started

### 1. Prerequisites
- Node.js 18+ (LTS)
- MongoDB Atlas account (for production)
- Vercel account (for hosting)

### 2. Installation
```bash
npm install
```

### 3. Environment Configuration
Create a `.env.local` file with the following:
```env
MONGODB_URI=your_mongodb_connection_string
NEXT_PUBLIC_GA_ID=your_analytics_id
SENDGRID_API_KEY=your_email_api_key
```

### 4. Development Mode
```bash
npm run dev
```
